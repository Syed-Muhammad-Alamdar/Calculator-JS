
# Interactive Calculator with LocalStorage History

A clean, modern web application built with HTML, CSS, and JavaScript that performs core arithmetic operations and remembers user calculation history.

## 🚀 Features
* **Core Math Operations:** Add, Subtract, Multiply, and Divide.
* **Data Persistence:** Uses browser `localStorage` to save calculations so history isn't lost on page refresh.
* **History Management:** Includes options to view and clear saved calculation history.
* **Modern UI:** Styled with a vibrant, clean card layout, CSS linear gradients, and a fully centered layout using Flexbox.

## 🛠️ Tech Stack
* HTML5
* CSS3 (Flexbox & Custom Styling)
* JavaScript (DOM Manipulation & LocalStorage API)
